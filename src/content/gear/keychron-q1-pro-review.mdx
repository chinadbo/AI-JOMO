## 购买动机

作为一个每天码字 8 小时以上的程序员，我对键盘的要求很简单：**好用、好看、好修**。

用过几十把键盘后，我总结出自己的理想配置：
- 75% 配列（保留 F 区和方向键，节省桌面空间）
- 铝坨坨（质感好，声音好）
- 无线（桌面整洁）
- 可编程（VIA/QMK 支持）
- 价格 1000 元左右（性价比优先）

Keychron Q1 Pro 正好满足所有要求。

## 产品规格

## 开箱体验

包装很扎实，黑色硬盒，打开后有：
- Q1 Pro 键盘本体
- USB-A to USB-C 线（带屏蔽磁环）
- USB-C to USB-C 线
- Mac/Win 键帽替换包
- 拔轴拔键器
- 螺丝刀（用于拆装）
- 说明书

**第一印象**：重。1.8kg 的铝坨坨，拿在手里很有分量。

## 外观设计

### 外壳

全铝合金 CNC 加工，表面阳极氧化处理。我选的是**黑色**，低调沉稳，桌搭百搭。

边框设计简洁，正面只有右下角小小的 Keychron  logo，不突兀。

### 配列

75% 配列是我最喜欢的配列之一：
- 保留了 F 区（F1-F12），写代码常用
- 保留了方向键和 PgUp/PgDown
- 删除了数字小键盘，节省桌面空间
- 右侧有旋钮（默认音量控制，可编程）

这个配列在**功能性和桌面占用**之间取得了完美平衡。

### 灯光

RGB 背光，支持多种灯效。不过铝坨坨的遮光性好，侧边看光效并不明显。

我个人日常关灯使用，省电且低调。

## 轴体选择

Q1 Pro 提供多种轴体选择，我选的是 **Keychron K Pro 红轴**。

### 轴体参数

| 参数 | 数值 |
|------|------|
| 触发压力 | 45g |
| 触底压力 | 60g |
| 触发行程 | 2.0mm |
| 总行程 | 4.0mm |
| 类型 | 线性轴 |

### 使用感受

**手感**：顺滑，没有明显段落感，适合快速打字。

**声音**：出厂有轻微弹簧音和弹片音，润滑后改善明显。

**对比**（我手头的其他轴体）：
- **Cherry 红轴**：K Pro 红更顺滑，Cherry 有点涩
- **Gateron 黄轴**：K Pro 红轻一些，黄轴更闷
- **TTC 快银**：快银触发更快，适合游戏，打字容易误触

**结论**：K Pro 红是不错的入门线性轴，日常使用足够。进阶玩家可以换金粉轴或快银。

## 改造过程

客制化的乐趣在于**折腾**。我对 Q1 Pro 做了以下改造：

### 1. 加声音包

虽然 Q1 Pro 出厂有夹心棉和底棉，但空格键和退格键还是不够扎实。

我加了一层 **PORON 轴下垫**，声音更集中，触底更干脆。

**成本**：¥30
**难度**：⭐（打开上盖，铺上去就行）
**效果**：⭐⭐⭐⭐ 声音明显改善

### 2. 润滑卫星轴

大键（空格、退格、回车、Shift）出厂有润滑，但不够充分。

我用 **Krytox 205g0** 重新润滑了卫星轴钢丝和轴心。

**成本**：¥50（润滑脂用不完）
**难度**：⭐⭐⭐（需要拆解、清洁、涂脂）
**效果**：⭐⭐⭐⭐⭐ 大键手感质的飞跃

### 3. 换键帽

原厂 ABS 双色键帽手感不错，但容易打油。

我换成了 **Akko ASA 高度 PBT 键帽**，灰色系，和黑色键盘很搭。

**成本**：¥199
**难度**：⭐（拔下来，按上去）
**效果**：⭐⭐⭐ 主要是颜值提升，手感变化不大

### 4. 贴特氟龙胶带

在 PCB 和定位板之间贴了一层特氟龙胶带，消除空腔音。

**成本**：¥10
**难度**：⭐⭐
**效果**：⭐⭐⭐ 声音更扎实

### 改造总成本

¥289，键盘手感从 80 分提升到 95 分。

## 声音测试

**改造前**：
- 空格键：钢丝音明显，松散
- 字母键：有轻微空腔音
- 整体：有点散，不集中

**改造后**：
- 空格键：扎实，闷脆声
- 字母键：集中，石子音
- 整体：麻将音，很解压

> 打字音视频可以在我 YouTube 频道观看（如果有的话）。

## 无线连接

### 蓝牙

Q1 Pro 支持蓝牙 5.1，最多配对 3 台设备（Fn+1/2/3 切换）。

**延迟**：日常使用感受不到延迟，打字跟手。
**稳定性**：3 个月使用中，断连次数 < 5 次，自动重连很快。
**续航**：我每天使用 8 小时，关灯使用，约 2 个月充一次电。

### 有线

USB-C 接口在左侧，走线方便。

线材质量不错，带屏蔽磁环，减少干扰。

## 编程功能

### VIA 支持

Q1 Pro 原生支持 VIA，浏览器打开 [usevia.app](https://usevia.app) 就能改键。

**我的改键方案**：
- 旋钮：按下 = 静音，旋转 = 音量
- Caps Lock → Escape（vim 党必备）
- Fn+Esc → 任务管理器
- Fn+WASD → 方向键（左手不用移开就能操作）

### 层（Layer）

Q1 Pro 支持 4 个层：
- 层 0：Mac 模式
- 层 1：Win 模式
- 层 2：自定义功能层
- 层 3：自定义功能层

我在层 2 设置了多媒体控制：
- Fn+Z/X/C = 上一首/播放暂停/下一首
- Fn+S/D = 屏幕亮度减/加

### 宏（Macro）

可以录制宏，一键执行复杂操作。

我录了一个「代码提交宏」：
- Fn+Tab → 自动输入 `git add -A && git commit -m ""`，光标移到引号内

## 优缺点总结

### ✅ 优点

1. **性价比极高**：千元价位铝坨坨 + 无线 + 热插拔 + QMK，无敌
2. **做工扎实**：CNC 铝合金，质感一流
3. **无线稳定**：蓝牙延迟低，续航长
4. **可玩性高**：热插拔、开源固件、声音包可改造
5. **兼容性好**：Mac/Win 无缝切换，快捷键支持完善

### ❌ 缺点

1. **重量大**：1.8kg，不适合频繁携带
2. **大键出厂一般**：需要手动润滑才能达到最佳状态
3. **ABS 键帽**：容易打油，建议换 PBT
4. **没有 2.4G**：只有蓝牙，对电竞玩家不够友好
5. **灯光一般**：铝坨坨遮光性好，RGB 效果不明显

## 长期使用感受（3 个月）

### 耐用性

外壳没有划痕，阳极氧化层很耐磨。

键帽换了 PBT，没有打油问题。

轴体顺滑度保持得很好，没有明显变涩。

### 续航

2 个月一充，符合官方标称。

### 稳定性

蓝牙连接稳定，偶尔断连会自动恢复。

这是我用过最满意的键盘，没有之一。
