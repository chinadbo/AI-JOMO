## 我的眼睛报警了

2023 年冬天，我的眼睛开始抗议。

连续几周的深夜加班后，我出现了这些症状：
- 晚上看屏幕时眼睛酸涩
- 早晨起床时畏光流泪
- 白天看东西有轻微重影

眼科医生的诊断很直接：「用眼过度，光线环境差。你需要改善照明。」

当时我的桌面照明方案：
- 吸顶灯（距离远，亮度不够）
- 台灯（占桌面，屏幕反光）
- 屏幕亮度最高（刺眼且不均匀）

**问题根源**：屏幕与环境的亮度对比过大，光源有直射眩光。

我开始了寻找完美桌面照明的旅程，三年间用了 4 款产品，最终定格在 BenQ ScreenBar Halo。

## 屏幕挂灯进化史

### 第一代：小米屏幕挂灯（¥199）

**优点**：
- 价格便宜
- 无线遥控
- 基本功能齐全

**缺点**：
- 光路控制一般，屏幕有轻微反光
- 亮度分布不均匀
- 色温调节档位少

**结论**：入门之选，够用但不完美。

### 第二代：明基 ScreenBar Plus（¥999）

**升级点**：
- 光路设计明显更好，屏幕无反光
- 显色指数高（Ra95）
- 有线控制，调节更精准

**缺点**：
- 有线控制器占桌面
- 没有背部照明
- 价格较高

**结论**：专业之选，但缺少氛围照明。

### 第三代：Yeelight 屏幕挂灯 Pro（¥399）

**尝试理由**：
- 支持 RGB 氛围灯
- 可以接入米家
- 价格便宜

**问题**：
- 主灯光路设计不如明基
- RGB 灯在屏幕后方，效果有限
- 做工一般

**结论**：功能多但不精，半年后背光坏了。

### 第四代：BenQ ScreenBar Halo（¥1,099）

**为什么最终选它**：
- 光路设计业界顶级
- 无线控制 + 背部氛围灯
- 前后灯独立控制
- 自动调光

这是我目前的主力照明，用了 18 个月，非常满意。

## 设计细节

### 光路设计（核心优势）

```
传统台灯的光路：
    ╲  ╱
     ╲╱        ← 光线直射眼睛，屏幕反光
    ┌──┐
    │  │
    └──┘

ScreenBar Halo 的光路：
    ╱╲
   ╱  ╲       ← 非对称光路，只照桌面
  ╱    ╲         不照屏幕，不直射眼睛
┌────────┐
│        │
└────────┘
```

**实际效果**：
- 屏幕零反光
- 桌面均匀照明
- 无直射眩光

这是 BenQ 的专利技术，也是它比其他品牌贵的原因。

### 重力枢轴夹

安装方式很聪明：

1. 把挂灯放到屏幕顶部
2. 调节配重块位置
3. 自动夹紧，无需工具

**适配性**：
- 支持 0.7-6cm 厚度的屏幕
- 平面屏和曲面屏都可用
- 不会遮挡摄像头

### 无线控制器

圆形旋钮设计，质感很好：

**操作逻辑**：
- 旋转：调节亮度/色温
- 按下：切换前后灯
- 长按：开启自动调光
- 双击：切换到「我的最爱」模式

**亮点**：
- 无线设计，桌面更整洁
- 内置光线传感器，支持自动调光
- 可以放置在顺手的位置

## 使用体验

### 场景一：深夜写代码

**设置**：
- 主灯：暖白光（4000K），60% 亮度
- 背灯：暖光（3000K），30% 亮度

**感受**：
- 桌面明亮但不刺眼
- 屏幕无反光，代码清晰可见
- 背灯减少屏幕与环境的亮度对比

**对比**：
以前用吸顶灯时，屏幕亮度需要调到最高才能看清，眼睛很累。
现在屏幕亮度可以调低，眼睛舒适度明显提升。

### 场景二：视频会议

**设置**：
- 主灯：冷白光（5000K），80% 亮度
- 背灯：关闭

**效果**：
- 面部补光均匀，无阴影
- 光线自然，不会过黄或过蓝
- Ra95 显色指数，肤色还原好

同事反馈：「你最近视频里的画面质量好了很多。」

### 场景三：阅读纸质书

**设置**：
- 主灯：暖光（3000K），100% 亮度

**效果**：
- 书本照明充足（>500 lux）
- 无频闪，长时间阅读不疲劳
- 高显色指数，图片色彩真实

### 场景四：氛围照明

**设置**：
- 主灯：关闭
- 背灯：暖光，低亮度

**场景**：
- 看电影时作为背景光
- 睡前放松时营造氛围
- 配合智能家居场景

**效果**：
柔和的间接照明，不会刺激眼睛，又能提供足够的夜间导航亮度。

## 功能详解

### 自动调光

控制器内置光线传感器，可以检测环境亮度，自动调节主灯亮度。

**使用建议**：
- 白天自然光强时，自动调低亮度
- 晚上环境暗时，自动提高亮度
- 保持桌面照度恒定在 500 lux 左右

**实际体验**：
自动调光很灵敏，从下午工作到晚上，几乎感受不到光线变化，但眼睛始终舒适。

### 我的最爱模式

可以保存一组灯光设置，双击控制器快速切换。

**我的配置**：

| 模式 | 主灯 | 背灯 | 使用场景 |
|------|------|------|----------|
| 专注 | 5000K/80% | 关闭 | 白天工作 |
| 舒适 | 4000K/60% | 3000K/30% | 晚上写代码 |
| 阅读 | 3000K/100% | 关闭 | 看书 |
| 氛围 | 关闭 | 2700K/20% | 放松/观影 |

### 前后灯独立控制

这是 Halo 相比 Plus 最大的升级。

**背灯的作用**：
1. **减少对比眩光**：屏幕亮，背景暗，眼睛容易疲劳。背灯提供背景照明，降低对比度。
2. **氛围营造**：色温可调，营造不同的工作和休息氛围。
3. **夜间导航**：起夜时不需要开大灯，背灯提供足够的照明。

**实际效果**：
开启背灯后，眼睛的舒适度明显提升，特别是晚上长时间工作时。

## 使用技巧

### 最佳安装位置

- 挂灯中心对准屏幕中心
- 挂灯前沿距离屏幕顶部 1-2cm
- 确保不会遮挡摄像头

### 亮度调节建议

```
白天（有自然光）：
- 主灯：60-80%
- 背灯：关闭或 20%

晚上（无自然光）：
- 主灯：50-70%
- 背灯：30-50%

深夜（环境很暗）：
- 主灯：40-60%
- 背灯：50-70%（减少对比眩光）
```

### 色温选择

- **4000-5000K**：专注工作（类似自然光）
- **3000-4000K**：一般使用（舒适平衡）
- **2700-3000K**：放松/睡前（促进褪黑素分泌）

**建议**：晚上避免使用纯冷白光（>6000K），会影响睡眠。

### 与其他灯光配合

**理想桌面照明方案**：

```
主照明：ScreenBar Halo
    ├── 提供桌面工作照明
    └── 减少屏幕眩光

环境照明：吸顶灯/落地灯（可调光）
    ├── 提供整体环境亮度
    └── 亮度约为 ScreenBar 的 30-50%

氛围照明：灯带/台灯
    └── 装饰和心情调节
```

**关键**：不要让屏幕成为环境中最亮的光源。

## 缺点与改进建议

### 缺点

1. **价格较高**：¥1,099 对很多人来说不便宜
2. **无 RGB**：想要炫酷灯效的人会失望
3. **无智能联动**：不支持 HomeKit/米家等智能家居
4. **背灯位置固定**：不能调节角度

### 改进建议

1. 推出 Lite 版本，去掉背灯，价格降到 ¥699
2. 增加 RGB 版本（虽然我不是目标用户）
3. 加入智能联动功能
4. 背灯角度可调
