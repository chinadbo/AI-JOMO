## 鼠标为什么重要

作为每天握持 8 小时以上的工具，鼠标对程序员的重要性常被低估。

直到 2022 年，我还在用买电脑送的普通鼠标。手腕酸痛、滚动长文件时手指抽筋、没有侧键快捷操作... 我以为这些都是「正常」的。

直到我换了 MX Master 3S。

那一刻的感受：**原来鼠标可以这么好用。**

两年后的今天，这款鼠标依然是我桌面上不可替代的工具。这篇文章分享我的深度使用体验。

## 产品定位

MX Master 系列是罗技的**旗舰办公鼠标**，3S 是 2022 年推出的升级款：

| 型号 | 年份 | 主要区别 |
|------|------|----------|
| MX Master 2S | 2017 | 基础款，已停产 |
| MX Master 3 | 2019 | USB-C 充电、MagSpeed 滚轮 |
| **MX Master 3S** | 2022 | 静音点击、8000 DPI、更精准 |
| MX Master 3 for Mac | 2022 | 针对 Mac 优化，USB-C to USB-C |

**目标用户**：
- 程序员、设计师、创意工作者
- 多设备用户（Mac + Windows）
- 需要高效率操作的知识工作者

## 核心功能详解

### 1. MagSpeed 电磁滚轮

**这是 MX Master 3S 的杀手级功能。**

传统滚轮机械结构，滚动时有段落感，速度慢。MagSpeed 采用电磁技术：

**两种模式**：
1. **逐行模式（Ratchet）**：有阻尼感，精准定位
2. **自由模式（Free Spin）**：无阻尼，一滑千行

**自动切换**：
- 慢速滚动 → 逐行模式
- 快速拨动 → 自动切换到自由模式，惯性滚动
- 点击滚轮 → 手动切换模式

**实际体验**：

| 场景 | 传统鼠标 | MX Master 3S |
|------|----------|---------------|
| 滚动 1000 行代码 | 手指抽筋 | 1 秒滑到底 |
| 精确选择代码块 | 容易滑过头 | 逐行模式精准 |
| 浏览长文档 | 痛苦 | 享受 |
| 看 Excel 大数据表 | 绝望 | 流畅 |

**声音**：
- 逐行模式：轻微机械声（比上一代静音 90%）
- 自由模式：几乎无声

**横向滚轮**：

拇指位置的横向滚轮，用于：
- Excel 横向滚动
- 时间轴横向移动（视频剪辑）
- 浏览器标签页切换

### 2. 8000 DPI 传感器

**升级点**：3 代是 4000 DPI，3S 升级到 8000 DPI

**意义**：
- 更高精度，适合高分辨率显示器（4K/5K）
- 玻璃表面也能使用
- 追踪更精准

**实际使用**：
我在 LG 34WK95U（5K2K 分辨率）上使用，DPI 设置 3200：
- 指针移动流畅不飘忽
- 精细操作（如视频剪辑）精准

**DPI 切换**：
通过 Logi Options+ 软件，可以设置 DPI 档位（200-8000）

### 3. 静音点击

**3S 的核心升级**：点击噪音降低 90%

**分贝对比**：
- 普通鼠标点击：约 45dB
- MX Master 3S：约 30dB

**场景价值**：
- 开放式办公室：不打扰同事
- 深夜工作：不吵醒家人
- 视频会议：点击声不会传入麦克风

**手感**：
- 点击反馈清晰，不绵软
- 键程适中，确认感强
- 静音但不失手感

### 4. 手势按钮

**设计**：拇指下方的隐形按钮，按住 + 移动鼠标 = 手势操作

**默认手势**：
| 手势 | 功能 |
|------|------|
| 按住 + 上滑 | Mission Control（显示所有窗口） |
| 按住 + 下滑 | 应用 Exposé（显示当前应用所有窗口） |
| 按住 + 左滑 | 桌面间左切换 |
| 按住 + 右滑 | 桌面间右切换 |

**自定义**：
在 Logi Options+ 中，可以为不同应用设置不同手势：
- VS Code：按住 + 左滑 = 上一个文件，右滑 = 下一个文件
- Chrome：按住 + 上滑 = 新建标签页，下滑 = 关闭标签页
- Figma：按住 + 移动 = 平移画布

**学习曲线**：
- 第 1 周：不习惯，容易误触
- 第 2 周：开始依赖
- 第 3 周：离不开

### 5. Flow 跨设备控制

**功能**：一套鼠标控制多台电脑，跨设备复制粘贴

**我的使用场景**：
- 主力：MacBook Pro（开发）
- 副机：Windows PC（测试/游戏）

**操作**：
1. 鼠标移动到屏幕边缘
2. 自动切换到另一台电脑
3. 无缝控制，像在用同一台电脑

**跨设备复制粘贴**：
- Mac 上复制一段代码
- 移动到 Windows 屏幕
- 直接粘贴，文件也可以

**技术要求**：
- 两台电脑在同一 WiFi 网络
- 都安装 Logi Options+ 软件
- 都配对同一个鼠标

**限制**：
- 不支持 Linux（Logi Options+ 无 Linux 版）
- 偶尔切换延迟 1-2 秒
- 复制大文件较慢

### 6. 多设备连接

**支持 3 台设备**：
- 2.4G（USB 接收器）
- 蓝牙 1
- 蓝牙 2

**切换方式**：
鼠标底部按钮一键切换，或 Flow 自动切换

**使用建议**：
- 主力电脑：2.4G（延迟最低）
- 平板/iPad：蓝牙 1
- 笔记本：蓝牙 2

### 7. 自定义按键

**总按键数**：7 个可编程按键

| 按键 | 默认功能 | 我的自定义 |
|------|----------|-----------|
| 左键 | 点击 | 保持默认 |
| 右键 | 右键菜单 | 保持默认 |
| 中键（滚轮按下） | 切换滚轮模式 | Mission Control |
| 后退键 | 浏览器后退 | IDE 上一个光标位置 |
| 前进键 | 浏览器前进 | IDE 下一个光标位置 |
| 手势按钮 | 手势 | 保持默认 |
| 滚轮左倾 | 横向滚轮左 | 上一个桌面 |
| 滚轮右倾 | 滚轮右 | 下一个桌面 |

**应用特定设置**：
Logi Options+ 支持为每个应用独立配置：

```
全局设置
├── VS Code
│   ├── 后退键：Go Back
│   ├── 前进键：Go Forward
│   └── 中键：Toggle Terminal
├── Chrome
│   ├── 后退键：Back
│   ├── 前进键：Forward
│   └── 中键：New Tab
├── Figma
│   └── 手势按钮：Pan Canvas
└── 其他应用继承全局设置
```

### 8. 快充与续航

**电池**：500mAh 内置锂电池

**官方续航**：
- 充满电可用 70 天
- 充电 1 分钟使用 3 小时

**实际体验**：
- 每天 8-10 小时使用
- 实际续航约 2 个月
- 充电 10 分钟可用一整天

**充电提示**：
- 电量低时 LED 指示灯变红
- 软件弹窗提醒
- 边充边用，无干扰

## 长期使用体验（2 年）

### 耐用性

**外观**：
- 橡胶侧握无磨损、无发粘
- 金属滚轮无掉漆
- 底部脚垫轻微磨损，不影响使用

**功能**：
- 滚轮顺滑如初
- 按键无双击现象
- 电池续航无明显衰减

### 软件体验

**Logi Options+**：
- 界面现代化，设置直观
- 偶尔启动较慢
- 需要登录账号（可离线使用基础功能）

**问题与解决**：
- **问题**：软件偶尔丢失设置
- **解决**：导出配置备份，重新导入

### 清洁维护

**日常清洁**：
- 每周用湿巾擦拭
- 滚轮缝隙用牙签清理
- 传感器孔保持干净

**橡胶侧握保养**：
- 避免手油/护手霜直接接触
- 偶尔用酒精棉片清洁

## 优缺点总结

### ✅ 优点

1. **MagSpeed 滚轮**：一旦用过，回不去
2. **人体工学**：长时间使用手腕不酸
3. **静音点击**：办公环境神器
4. **Flow 跨设备**：多电脑用户福音
5. **高 DPI**：高分辨率屏幕流畅
6. **续航持久**：两个月充一次
7. **快充**：应急充电几分钟用一天
8. **软件丰富**：按键自定义程度极高

### ❌ 缺点

1. **价格贵**：¥899，比普通鼠标贵 10 倍
2. **体积大**：小手用户可能不适
3. **重量重**：141g，游戏玩家嫌重
4. **不适合游戏**：不是游戏鼠标，延迟不如有线
5. **Linux 支持差**：Flow 不可用，软件功能受限
6. **滚轮模式切换**：自动切换有时不准确
7. **侧键位置**：拇指短的按后退键略费劲

## 设置指南

### 到手必做设置

1. **下载 Logi Options+**
   - Mac/Windows 必装
   - 解锁全部功能

2. **设置滚轮行为**
   - SmartShift：启用（自动切换滚轮模式）
   - 灵敏度：根据个人习惯调整

3. **配置按键**
   - 设置应用特定配置
   - 导出配置备份

4. **开启 Flow**（多设备用户）
   - 所有设备安装 Logi Options+
   - 配对同一鼠标
   - 开启 Flow 功能

### 我的配置文件分享

**VS Code 配置**：
```
后退键：Go Back（Alt+Left）
前进键：Go Forward（Alt+Right）
中键：Toggle Terminal（Ctrl+`）
手势上：Show All Symbols（Ctrl+Shift+O）
```

**Chrome 配置**：
```
后退键：Back（Alt+Left）
前进键：Forward（Alt+Right）
中键：New Tab（Ctrl+T）
手势上：New Window（Ctrl+N）
手势下：Close Tab（Ctrl+W）
```

## 总结

MX Master 3S 是我用过最好的办公鼠标，没有之一。

它不是完美的（价格贵、不适合游戏），但在**办公效率**这个核心场景上，它做到了极致。

MagSpeed 滚轮改变了我和代码交互的方式，Flow 让多设备工作流无缝衔接，静音点击让深夜工作不再愧疚。

如果你每天使用鼠标超过 4 小时，MX Master 3S 是一笔值得的投资。

> 所谓「终点」，不是不需要更好的鼠标，而是不再想换其他鼠标。
