## 为什么桌面配置很重要

作为一个每天要在电脑前度过 10+ 小时的软件工程师，我越来越相信一个观点：

> **你的工作环境决定了你的工作质量。**

三年前，我的"桌面"就是一张宜家便宜桌子 + 笔记本 + 随机买的椅子。结果是：颈椎痛、眼睛疲劳、效率低下。

现在，经过 3 年迭代，我的桌面已经成为一个**为深度工作优化的系统**。每个设备、每个摆放位置都有其目的。

这篇文章分享我的 2026 桌面配置，以及背后的设计思路。

**核心原则**：所有高频使用的物品都在手臂范围内，零低头、零侧身。

## 显示系统：LG 34WK95U 5K 超宽屏

### 为什么选择超宽屏？

我曾经用双屏（27" + 27"），但有几个问题：
- 中间有边框，视线需要跳跃
- 占用的桌面空间太大
- 两个屏幕色温/亮度不一致

34 英寸 21:9 超宽屏（5120×2160）完美解决这些问题：
- **单屏无缝**：没有恼人的边框
- **5K 分辨率**：文字锐利，代码清晰
- **分区工作**：左边代码，右边文档/浏览器
- **色彩一致**：单屏色温亮度统一

### 窗口管理

我用 [Rectangle](https://rectangleapp.com) 管理窗口布局：

| 快捷键 | 布局 |
|-------|------|
| `⌃⌥←` | 左半屏 (50%) |
| `⌃⌥→` | 右半屏 (50%) |
| `⌃⌥↑` | 上半屏 |
| `⌃⌥↓` | 下半屏 |
| `⌃⌥U` | 左上 1/4 |
| `⌃⌥I` | 右上 1/4 |
| `⌃⌥J` | 左下 1/4 |
| `⌃⌥K` | 右下 1/4 |
| `⌃⌥↩` | 最大化 |

**典型工作布局**：
- 左侧 60%：VS Code / Cursor
- 右侧 40%：Chrome（文档 + ChatGPT）+ iTerm2（上下分屏）

### 护眼设置

- **亮度**：根据环境光自动调节（配合光线传感器）
- **色温**：白天 6500K，晚上 4000K（Night Shift）
- **距离**：眼睛到屏幕 60-70cm
- **高度**：屏幕上沿与眼睛平齐或略低

## 输入设备

### 键盘：Keychron Q1 Pro

已经在之前的[评测文章](/gear/keychron-q1-pro-review)中详细介绍过。

**简要总结**：
- 75% 配列，节省桌面空间
- 铝坨坨，打字声音好听
- 无线连接，桌面整洁
- QMK/VIA 支持，高度可编程

**自定义快捷键**（通过 VIA）：
- `Fn + Q`：打开 Cursor
- `Fn + W`：打开 Chrome
- `Fn + E`：打开 iTerm2
- `Fn + R`：打开 Notion
- `Fn + S`：截图
- `Fn + D`：勿扰模式切换

### 鼠标：Logitech MX Master 3S

**为什么选它**：
- **MagSpeed 滚轮**：一秒滚动千行代码
- **侧向滚轮**：Excel/表格必备
- **手势按钮**：按住侧键 + 移动 = 多任务切换
- **Flow 功能**：跨 Mac/Windows 无缝切换

**自定义设置**（Logi Options+）：
- 滚轮按下：Mission Control
- 侧键前进/后退：代码跳转（配合 IDE）
- 手势上：显示桌面
- 手势下：应用 exposé

### 鼠标垫：Pulsar ParaControl V2

- 尺寸：900×400mm，覆盖键盘+鼠标区域
- 表面：细面，适合精准操作
- 底部：防滑橡胶，稳固不移动

## 音频系统

### 耳机：Sony WH-1000XM5

**使用场景**：
- 深度工作时（降噪开启）
- 会议通话（麦克风清晰）
- 通勤/出差（便携折叠）

**降噪等级设置**：
- 专注模式：最大降噪（几乎听不到外界）
- 环境声模式：听到人声（适合办公室）
- 关闭：省电

### 桌面音箱：Audioengine A2+

**使用场景**：
- 休闲时听音乐/播客
- 不需要深度专注的背景音
- 不想戴耳机时

**摆放**：显示器两侧，45° 角指向耳朵

### 音频切换

用 [SoundSource](https://rogueamoeba.com/soundsource/) 快速切换音频输出：
- `⌃⌥⌘↑`：耳机
- `⌃⌥⌘↓`：音箱

## 计算设备

### 主力：MacBook Pro 14" (M3 Pro)

**配置**：
- M3 Pro 芯片 (12核 CPU, 18核 GPU)
- 36GB 统一内存
- 1TB SSD

**使用方式**：
- 闭合状态放在 [Rain Design mStand](https://www.raindesigninc.com/mstand.html) 支架下
- 外接显示器作为主屏
- 需要移动时直接拔线带走

**为什么选 14" 而不是 16"**：
- 外接显示器时屏幕大小不重要
- 14" 更便携，出差无压力
- 散热足够，性能释放充分

### 充电：Anker 737 充电器 + 磁吸无线充

- **Anker 737**：140W GaN 充电器，同时充笔记本+iPhone+AirPods
- **磁吸无线充**：桌面随手放，随时满电

### 台灯：BenQ ScreenBar Halo

**为什么选它**：
- 挂在显示器上，不占用桌面空间
- 非对称光路，屏幕不反光
- 无线控制器，调节亮度色温
- 背部氛围灯，减少眼部疲劳

**使用模式**：
- 白天：冷白光 100%（提神）
- 傍晚：暖白光 70%（过渡）
- 深夜：暖黄光 40%（护眼）

### 线缆管理

**桌面下**：
- [宜家 SIGNUM](https://www.ikea.com/cn/zh/p/signum-ikea-903-002-63/) 线槽收纳电源适配器
- 魔术贴绑带固定线缆
- 桌下粘贴式线扣引导走向

**桌面上**：
- 尽量无线设备（键盘、鼠标、耳机）
- 仅剩：显示器线、充电线（磁吸，不用时自动收回）

**效果**：从正面看，几乎看不到任何线缆。

## 座椅：Herman Miller Aeron

**投资回报率最高的装备**。

我曾经用几百块的椅子，结果：
- 坐 2 小时就腰酸背痛
- 夏天背部出汗
- 椅子越坐越松垮

Aeron 解决所有问题：
- **Pellicle 网布**：透气，夏天不闷
- **PostureFit SL**：腰部支撑到位
- **全可调**：座高、扶手、后仰张力都能调
- **耐用**：12 年保修，用十年没问题

**正确坐姿**：
- 脚平放地面，大腿与地面平行
- 背部贴紧椅背，腰部有支撑
- 手肘 90°，手腕自然放在键盘上
- 眼睛平视屏幕上沿

## 环境设计

### 照明

**原则**：多层次照明，避免单一光源

1. **顶灯**：Philips Hue 智能灯，2700K-6500K 可调
2. **台灯**：BenQ ScreenBar Halo，局部照明
3. **背景灯**：显示器背后 LED 灯带，减少对比眩光
4. **自然光**：窗帘透光率 50%，白天利用自然光

**智能场景**（通过 HomeKit）：
- "开始工作"：台灯 100% 冷白，顶灯 50%，背景音乐播放专注歌单
- "深度工作"：台灯 80% 暖白，顶灯关闭，勿扰模式开启
- "休息"：台灯 40% 暖黄，背景灯开启，播放轻松音乐

### 空气质量

- **空气净化器**：小米净化器 4 Pro，保持 PM2.5 < 35
- **加湿器**：Dyson AM10，保持湿度 40-60%
- **绿植**：桌上小盆绿萝，缓解视觉疲劳

### 温度

- **空调**：25°C 恒温
- **桌面风扇**：Noctua NF-A12，夏天局部降温
- **保暖**：冬天桌下暖脚器

## 总结

好的桌面配置不是炫耀装备，而是**设计一个让自己进入心流的环境**。

核心原则：
1. **身体舒适**：椅子、显示器高度、键盘位置
2. **视觉整洁**：无线化、线缆管理、简约风格
3. **听觉沉浸**：降噪、背景音乐
4. **智能自动化**：一键切换工作模式
5. **逐步迭代**：按需升级，不一步到位

投资你的工作环境，就是投资你的工作效率和健康。
